How this works: a GitHub Actions workflow runs scraper.py on a schedule using a real headless browser, checks each tracked page, and commits the results to listings.json in this repo. This page — hosted on GitHub Pages — reads that file and alerts you here. No server of your own required. See README.md in the repo for setup.
